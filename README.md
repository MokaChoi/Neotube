# Neotube
- css, html 연습으로 유튜브 사이트를 클론코딩하였습니다

## 기능
1. 기본적인 영상 재생
2. 웹페이지 화면크기에 따라 바뀌는 구조
3. 글자 길이에 따른 글자 토글버튼 활성화
4. 웹페이지를 스크롤해도 화면은 위에 고정됨
